Step 1.
Compile the application (build) FileSearchLib

Step 2.
Right-click the "FileSearchApp" project in Solution Explorer.
Select "Properties" or "Properties".
In the application properties window, open the "C/C++" -> "General" tab.
In the "Additional Include Directories" field, add the path to the directory containing the header files  "FileSearchLib" static library.

Step 3.
Compile the application (build) FileSearchApp
After successfully building "FileSearchApp", you will get an executable file named "FileSearchApp.exe".
You can run it from the command line to search for a file on localhost.
